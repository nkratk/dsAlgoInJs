Queue

  • FIFO - First in first out Ex: Grocery store
  • Enqueue, Dequeue
		class Queue {
			constructor () {
				this.list = [];
			}
			enQueue (val) {
			//add to last
				this.list.push(val);
				return this.list;
			}
			deQueue () {
				//remove from first
				this.list.shift();
				return this.list;
			}
			peek () {
				//head - get the first one
				return this.list[0];
			}
		}
